A modern Gutenberg block that provides a customizable language switcher for multilingual websites with Polylang integration. 

== Description ==

Language Switcher is a plugin that adds a language-switcher component to the WordPress Gutenberg editor.
Language Switcher requires that the 'Polylang' plugin is installed and activated. 
The Lanugae Switcher Block retrieves available translations for the current page and shows them in a list. 
If there are no available translations (when a page only exists in 1 language), the list is hidden.

== Installation ==

To install this plugin, you can download the zip file, and upload it through the WordPress Plugin page.

1. Download the plugin zip. 
1. Install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
